OpenCV之高斯滤波与中值滤波

发表于 2021-01-15 更新于 2021-01-19 分类于 opencv

高斯滤波

  • 高斯滤波是一种线性平滑滤波，适用于消除高斯噪声，广泛应用于图像处理的减噪过程
  • 高斯滤波是一个对整幅图像进行加权平均的过程，每一个像素点的值，都由其本身和领域内的其他像素值经过加权平均后得到
  • 用一个模板扫描图像中的每一个像素，用模板确定的领域内像素的加权平均灰度值去替代中心像素点的值。
  • 高斯滤波对于抑制服从正态分布的噪声非常有效

OpenCV中的高斯滤波

-OpenCV高斯滤波AOI函数如下：

GaussianBlur(
InputArray src,
OutputArray dst,
Size ksize,//ksize是高斯滤波窗口大小
double sigmaX,//X方向滤波系数
double sigmaY,//Y方向滤波系数
int borderType=BORDER_DEFAULT //默认边缘插值方法
)
  • 注意：当Size为（0,0）时，就会从sigmaX开始计算生成高斯卷积核系数，当Size不为0时，优先从Size开始

中值滤波

  • 中值滤波本质上是统计排序滤波器的一种，其对图像特定噪声类型（椒盐噪声）会取得比较好的去噪效果，也是比较常见的图像去噪声与增强的方法之一
  • 中值滤波是窗口在图像上移动，对其覆盖的对于ROI区域下的所有像素值排序，以中值作为中心像素点的输出值。一般用于去除椒盐噪声。

OpenCV中的中值滤波

-OpenCV中值滤波API函数如下：

medianBlur(
InputArray src
OutputArray dst
int ksize
)
Python：dst=cv.medianBlur(src,ksize[,dst])

-需要注意的是:其中的ksize必须是奇数

图片色彩空间介绍

  • 图片的色彩空间有很多种，如BGR、HSV、GRAY、yuv、ycrcb等，这里我们主要看RGB、HSV、GRAY
  • BGR色彩空间：BGR是通过蓝绿红三原色来描述颜色的色彩空间
  • HSV色彩空间：HSV是根据颜色的直观特性来描述的，也称六角椎体模型，这个模型中颜色的参数分别是色调(HSV)、饱和度(S)、亮度(V)
