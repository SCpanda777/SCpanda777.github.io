OpenCV中值滤波medianBlur

发表于 2021-01-15 分类于 opencv

中值滤波

  • 中值滤波本质上是统计排序滤波器的一种，其对图像特定噪声类型（椒盐噪声）会取得比较好的去噪效果，也是比较常见的图像去噪声与增强的方法之一
  • 中值滤波是窗口在图像上移动，对其覆盖的对于ROI区域下的所有像素值排序，去中值作为中心像素点的输出值。一般用于去除椒盐噪声。

OpenCV中的中值滤波

-OpenCV中值滤波API函数如下：

medianBlur(
InputArray src
OutputArray dst
int ksize
)
Python：dst=cv.medianBlur(src,ksize[,dst])

-需要注意的是:其中的ksize必须是奇数
